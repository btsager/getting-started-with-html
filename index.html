WEB & HTML Basics

The baiscs of the WEB are broken into 5 categories; WEB, HTML, URLS, HTTP, and WEB Applications. WEB interaction involves computers, HTTP requests, servers, HTML documents, and web browsers. The WWW is a collection of HTML documents, HTML stands for Hypertext Markup Language, it glues everything together and is the heart of the WEB. Hyperlinks link the HTML documents together, makign the WEB, web like. The Internet was invented in 1990, it has over 30 billion pages. This video gives a good overview.

HTML, TAGS, & ELEMENTS

HTML documents contain text content which describes "what you see", markup which describes "how it looks", reference to other documents images and videos, and links to other pages. HTML uses tags to tell browsers what type of HTML element to use. This video gives a good overview. HTTP Elements include the opening tag, content, and closing tag. Tags that have content require a closing tag, if the tag does not have content it is known as void and does not require a closing tag.

White Space, Inline, Block

The size of white space always equals one, no matter how large or long the white space appears in HTML. Two Tags that help break out or group the HTML content are line break and paragrah. In addiiton HTML Elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Break is inline and paragrah is block. This video explains the distinction well.

Main Take Away Topics
-What a Web Page is: A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.
-How Coding Works: Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote.
-Computers are Stupid: Programmers need to write exactly the way a computer understands (also known as writing with correct "syntax").
-Programmers Can't Remember Everything: There are too many details to keep everything in your head. And that's okay. If you forget how to make text italic in HTML, you can always just look it up.
-Basic HTML Vocabulary: Tag, Element, & Attribute
